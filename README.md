# My-Python-Pandas
# 🐍 Python Data Analysis Project
### Data Technician Bootcamp – Week 6  

---

## 🧾 Overview  
This project was completed as part of a **Data Technician Bootcamp**, focusing on applying **Python and Pandas for data analysis**.

The project involved working with **retail and sales datasets** to perform **data cleaning, transformation, and exploratory analysis**. Using Pandas and visualisation libraries, raw data was converted into meaningful insights through structured analysis and clear visual outputs.

---

## 🛠️ Tools & Technologies  
- **Python**  
- **Pandas** (Data Analysis)  
- **Matplotlib** (Data Visualisation)  
- Google Colab / Jupyter Notebook  

---

## 💡 Core Competencies  

### 🧪 Data Cleaning & Preparation  
- Handled missing data using:
  - `dropna()` – removing incomplete records  
  - `fillna()` – replacing missing values  
- Ensured data consistency and correctness before analysis  
- Prepared datasets for reliable insights and reporting  

---

### 🔍 Data Filtering & Subsetting  
- Filtered datasets to focus on relevant data points  
- Used:
  - `.loc[]` – label-based filtering  
  - `.iloc[]` – position-based filtering  
- Selected and extracted subsets of data for targeted analysis [1](https://b2wcompletetraining057-my.sharepoint.com/personal/bahjoawad_bootcamp_leepgroup_com/_layouts/15/Doc.aspx?sourcedoc=%7B138B2FB0-0408-4ED0-BD73-29170811490F%7D&file=Data_Technician_Workbook_Week_6%20-%202026.docx&action=default&mobileredirect=true)  

---

### 🔄 Data Manipulation & Transformation  
- Applied key Pandas methods to shape and analyse data:

  - `groupby()` – grouped data to calculate aggregated metrics  
  - `sort_values()` – organised data to identify trends and rankings  
  - Column operations – created and modified data fields  
  - Data aggregation – summarised performance across categories  

- Enabled efficient transformation of large datasets into structured insights  

---

### 📊 Data Visualisation  
- Created visualisations using:
  - Built-in Pandas plotting  
  - **Matplotlib**  

- Developed charts such as:
  - Bar charts (category comparisons)  
  - Line charts (trend analysis)  
  - Pie charts (proportion analysis)  

- Improved understanding of patterns and trends through visual outputs
